üåê Network & Internet

WiFi is not connecting or shows "No Internet" +
Try these steps:
  1. Forget and Reconnect:
    • Go to WiFi settings
    • Click on the network ‚Üí "Forget"
    • Search again and reconnect with password
  2. Restart WiFi: Turn WiFi off for 10 seconds, then back on
  3. Restart Device: Sometimes a simple restart fixes the issue
  4. Check if others have internet: If everyone has the problem, it's a network issue - call ICT
üí° Tip: Make sure you're connecting to the correct hospital WiFi network (WCSH-Staff or WCSH-Guest)
WiFi password - what is it? +
WiFi Credentials:
  • Staff Network: WCSH-Staff (Password: Contact ICT)
  • Guest Network: WCSH-Guest (Password: Contact Reception)
‚ö†Ô∏è Security Note: Never share the staff WiFi password with visitors or patients. Use the guest network for non-staff.

To get the password: Call ICT at +251-XXX-XXX-XXX or visit ICT office

Slow internet - pages taking too long to load +
Quick fixes for slow internet:
  1. Close unnecessary programs: Each program uses bandwidth
  2. Check who's connected: Too many devices on one network slows it down
  3. Move closer to WiFi router: Weak signal = slow speed
  4. Restart your router: Unplug for 30 seconds, plug back in
  5. Clear browser cache: Old files can slow browsing
‚úÖ Still slow? Report it to ICT - there might be a network issue we need to fix
Cannot access shared network drives +
Steps to fix network drive access:
  1. Check network connection: Make sure you're connected to hospital WiFi
  2. Restart computer: This refreshes network connections
  3. Re-map the drive:
    • Open File Explorer
    • Right-click "This PC" ‚Üí Map network drive
    • Enter drive path (contact ICT for correct path)
  4. Check permissions: You might not have access - contact ICT to grant permissions

üíª Computer Issues

Computer is very slow or freezing +
Speed up your computer:
  1. Close unused programs: Press Ctrl+Alt+Delete ‚Üí Task Manager ‚Üí End unnecessary tasks
  2. Restart regularly: Restart your computer at least once a day
  3. Check disk space: Right-click C: drive ‚Üí Properties (should have at least 20% free)
  4. Run disk cleanup: Search "Disk Cleanup" ‚Üí Select C: drive ‚Üí Clean up system files
  5. Disable startup programs: Task Manager ‚Üí Startup tab ‚Üí Disable programs you don't need
üí° Prevention: Don't install personal software. Only use approved hospital applications.
Computer won't turn on or shows black screen +
Troubleshooting steps:
  1. Check power:
    • Is the power cable plugged in?
    • Is the power outlet working? (Try another device)
    • Check if power button light is on
  2. Check monitor:
    • Is monitor turned on?
    • Is monitor cable connected to computer?
    • Try pressing monitor power button
  3. Hard restart: Hold power button for 10 seconds, wait 30 seconds, turn back on
‚úÖ Still not working? This needs ICT assistance - report it as HIGH priority
Cannot login - "Incorrect password" or account locked +
Login issues:
  1. Check Caps Lock: Make sure Caps Lock is OFF (passwords are case-sensitive)
  2. Check keyboard language: Make sure it's set to English
  3. Try your password slowly: Type carefully, character by character
  4. Account locked after 3 wrong attempts? Wait 30 minutes or call ICT to unlock
‚ö†Ô∏è Forgot Password? Call ICT immediately. Never share your password or write it down!
Need to install software or program +
Software Installation Policy:

For security reasons, regular users cannot install software. Here's what to do:

  1. Submit a request: Use the "Report Issue" form and select "Software Installation"
  2. Provide details:
    • Name of software needed
    • Purpose/reason for needing it
    • Department approval (if required)
  3. ICT will: Review the request, check security, and install if approved
üí° Common approved software: Microsoft Office, Adobe Reader, Chrome browser - these can usually be installed quickly

üñ®Ô∏è Printers & Scanners

Printer not printing or showing offline +
Quick printer fixes:
  1. Basic checks:
    • Is printer turned on? (Check power light)
    • Is there paper in the tray?
    • Are all cables connected?
    • Check for error messages on printer display
  2. Restart printer: Turn off, wait 30 seconds, turn back on
  3. Check printer queue:
    • Go to Control Panel ‚Üí Devices and Printers
    • Right-click printer ‚Üí "See what's printing"
    • Cancel stuck jobs (Printer ‚Üí Cancel All Documents)
  4. Set as default: Right-click printer ‚Üí "Set as default printer"
  5. Check "Use Printer Offline": Make sure it's NOT checked
Paper jam - how to clear it +
Clear paper jam safely:
  1. Turn printer OFF: Always turn off before clearing jams
  2. Open printer cover: Usually the front or top panel
  3. Remove paper CAREFULLY:
    • Pull in the direction paper feeds (usually forward)
    • Don't tear the paper - remove whole sheet
    • Check for small torn pieces
  4. Check paper tray: Make sure paper is loaded correctly (not too much!)
  5. Close cover and turn ON: Wait for printer to warm up
  6. Test print: Try printing a test page
üí° Prevention: Don't overload paper tray. Fan paper before loading. Use fresh, dry paper.
Print quality issues - faded, streaky, or blurry +
Improve print quality:
  1. Check ink/toner levels: Replace if low (printer display shows levels)
  2. Run printer cleaning:
    • Control Panel ‚Üí Devices and Printers
    • Right-click printer ‚Üí Printing Preferences
    • Find "Maintenance" or "Tools" tab
    • Run "Clean Print Heads" or "Nozzle Check"
  3. Check paper quality: Use proper printer paper, not copy paper
  4. Align print heads: Use printer's alignment tool (same location as cleaning)
‚úÖ Need new toner/ink? Report to ICT - we'll replace it
How to scan documents +
Scanning steps:
  1. Place document: Face down on scanner glass, align with corner markers
  2. On computer:
    • Open "Windows Fax and Scan" or printer software
    • Click "New Scan"
    • Select scanner from list
    • Choose format (PDF for documents, JPEG for images)
    • Click "Scan"
    • Save file to desired location

üìß Email & Communication

Cannot send or receive emails +
Email troubleshooting:
  1. Check internet connection: No internet = no email
  2. Check Outbox: Stuck emails prevent new ones from sending
  3. Verify email settings: Incorrect settings can block email
  4. Check mailbox size: Full mailbox blocks new emails
  5. Try webmail: Access email through web browser to test
üí° Tip: Regularly delete old emails and empty deleted items folder
How to setup email on phone +
Mobile email setup:

For Android:

  1. Open Email or Outlook app
  2. Add Account ‚Üí Exchange or Office 365
  3. Enter your hospital email address
  4. Enter your password
  5. Follow prompts to complete

For iPhone:

  1. Settings ‚Üí Mail ‚Üí Accounts ‚Üí Add Account
  2. Select Exchange
  3. Enter email and description
  4. Server: mail.wcsh.et (contact ICT for exact server)
  5. Enter username and password
‚úÖ Need help? Contact ICT with your phone model - we'll help you set it up
